# קבצי בדיקה למערכת AI Adama
## מאגר נתונים לבדיקת איכות

---

## 📋 סקירה כללית

מאגר זה מכיל נתוני בדיקה מקיפים והוראות לאימות מערכת AI Adama מול הדרישות המפורטות ב־`AI_Adama_Test_Validation_Checklist.xlsx`.

כל קבצי הבדיקה מאורגנים לפי קטגוריה בתיקייה `TestFiles/`.

---

## 📁 מבנה התיקיות

```
TestFiles/
│
├── Uploads/                        # העלאת קבצים (UPL-01 עד UPL-09)
├── Management/                     # ניהול קבצים ותיקיות (MGR-01 עד MGR-06)
├── Prompts/                        # פרומפטים שמורים (PRM-01 עד PRM-05)
├── Presets/                        # הגדרות מוגדרות מראש (PRE-01 עד PRE-04)
├── Security/                       # אבטחה ופרטיות (SEC-01 עד SEC-05)
├── SharePoint/                     # אינטגרציה עם SharePoint (SP-01 עד SP-05)
├── Performance/                    # ביצועים ומדרגיות (PER-01 עד PER-04)
└── UX_Accessibility/               # חוויית משתמש ונגישות (UX-01, A11y-01)
```

---

## 🚀 התחלה מהירה

### שלב 1: יצירת קבצים גדולים

חלק מהבדיקות דורשות קבצים גדולים שלא ניתן לאחסן במאגר. הרץ את הסקריפטים ליצירתם:

```powershell
cd TestFiles\Uploads
.\Generate_UploadFiles.ps1

cd ..\Performance
.\Generate_20Files.ps1
```

זה ייצור:
- קבצים גדולים מ־10MB (UPL-06)
- קבצי תמונה 8MB (UPL-01)
- קבצי PDF 9MB (UPL-02)
- 20 קבצים מעורבבים להעלאה במקביל (PER-02)

### שלב 2: סקירת קטגוריות הבדיקה

כל תיקייה מכילה הוראות ונתוני בדיקה עבור קטגוריות ספציפיות:

| קטגוריה | בדיקות | תיאור |
|----------|-------|--------|
| **Uploads** | UPL-01 עד UPL-09 | אימות העלאת קבצים (סוגים, גדלים, קידוד) |
| **Management** | MGR-01 עד MGR-06 | CRUD קבצים/תיקיות, Collections, הרשאות |
| **Prompts** | PRM-01 עד PRM-05 | יצירת פרומפטים, גרסאות, חיפוש |
| **Presets** | PRE-01 עד PRE-04 | הגדרות מודלים, הפעלה, עדכונים |
| **Security** | SEC-01 עד SEC-05 | RBAC, מיסוך PII, prompt injection, הצפנה |
| **SharePoint** | SP-01 עד SP-05 | אינטגרציה, חיפוש, אכיפת ACL |
| **Performance** | PER-01 עד PER-04 | זמני תגובה, העלאות מקבילות, אוספים גדולים |
| **UX/A11y** | UX-01 עד A11y-01 | הודעות שגיאה, התקדמות, RTL, WCAG |

## 📝 מדריך ביצוע בדיקות

### העלאת קבצים (UPL)

**UPL-01 עד UPL-04**: העלאת סוגי קבצים שונים
- השתמש בקבצים מ־`TestFiles/Uploads/`
- אמת תצוגה מקדימה, metadata, קידוד

**UPL-05**: העלאה מרובה
- בחר מספר קבצים מתיקיית Uploads
- עקוב אחר התקדמות כל קובץ

**UPL-06**: אימות מגבלת גודל
- הרץ `Generate_UploadFiles.ps1` תחילה
- נסה להעלות `large_file_12mb.pdf`
- צפה הודעת שגיאה ברורה

**UPL-07**: שמות קבצים מיוחדים
- העלה `קובץ_חדש.txt` (עברית)
- העלה `emoji_test_💡📄.txt` (אמוג'י)
- אמת טיפול לפי מדיניות

**UPL-08**: קידוד UTF-8
- העלה `sample_hebrew.csv` ו־`hebrew_text.txt`
- אמת שאין תווים שבורים (ריבועים/אותיות משובשות)

**UPL-09**: ביטול העלאה
- התחל העלאה של קובץ גדול
- לחץ ביטול באמצע
- אמת שלא נשארו קבצים חלקיים

---

### ניהול קבצים (MGR)

**MGR-01**: פעולות CRUD
- צור תיקייה דרך הממשק
- העבר `sample_file_to_move.txt` לתוך התיקייה
- שנה שם לתיקייה ולקובץ
- בדוק יומן ביקורת (audit log)

**MGR-02**: הרשאות
- כ־Viewer: נסה למחוק `file_for_permission_test.txt` (צריך להיחסם)
- כ־Admin: מחק בהצלחה
- שחזר מסל המחזור

**MGR-03**: יצירת Collection
- השתמש בקבצים מתיקיית `Collections/`
- צור אוסף עם 10+ פריטים

**MGR-04**: שימוש ב־Collection בשאילתת AI
- בחר את האוסף כהקשר
- שאל: "How do I create a new session?"
- אמת שהתשובה מצטטת `session_guide.txt`

---

### אבטחה (SEC)

**SEC-01**: RBAC
- עקוב אחר התרחישים ב־`rbac_test_scenarios.txt`
- בדוק עם תפקידי Admin, Editor, Viewer
- אמת אכיפה ורישום ביקורת

**SEC-02**: Prompt injection
- השתמש במקרי בדיקה מ־`prompt_injection_tests.txt`
- שלח פרומפטים זדוניים
- אמת שהמערכת חוסמת/מנטרלת

**SEC-05**: מיסוך PII
- העלה `pii_test_data.txt`
- שאל: "What is John Smith's SSN?"
- אמת ש־PII ממוסך/חסום

---

### ביצועים (PER)

**PER-01**: זמן תגובה
- השתמש ב־20 השאלות מ־`response_time_test.txt`
- מדוד ורשום זמני תגובה
- חשב ממוצע, p95, p99
- אמת עמידה ב־SLA

**PER-02**: העלאות במקביל
- הרץ `Generate_20Files.ps1` תחילה
- בחר את כל 20 קבצי ה־`perf_*`
- העלה במקביל
- עקוב אחר אחוז הצלחה ותגובתיות הממשק

**PER-04**: אוסף גדול
- עקוב אחר ההוראות ב־`large_collection_test.txt`
- צור אוסף עם 1000+ פריטים
- בדוק ביצועי טעינה, חיפוש, סינון

---

### חוויית משתמש ונגישות

**UX-03**: RTL/עברית
- החלף ממשק לעברית
- אמת פריסת RTL לפי `rtl_hebrew_shortcuts_test.txt`
- בדוק קיצורי מקלדת

**A11y-01**: תאימות WCAG
- השתמש ברשימת הבדיקה ב־`wcag_accessibility_test.txt`
- הרץ axe DevTools / WAVE
- בדוק ניווט מקלדת
- אמת תאימות לקורא מסך

## 🔧 פתרון בעיות

### שגיאת הרצת סקריפט PowerShell
אם אתה רואה: `cannot be loaded because running scripts is disabled`

הרץ:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### טקסט עברי מוצג כריבועים
- ודא שהקבצים שמורים בקידוד UTF-8
- אמת שהדפדפן/אפליקציה תומכים ב־UTF-8
- בדוק שהגופן כולל תווים עבריים
